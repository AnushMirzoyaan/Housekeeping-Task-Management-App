## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

This a simple mobile application for housekeeping department. 
The application will allow a housekeeper to manage cleaning tasks for our hotel apartments.
